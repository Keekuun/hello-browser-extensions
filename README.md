## 浏览器扩展开发框架-WXT

[WXT](https://github.com/wxt-dev/wxt) 是一个用于开发浏览器扩展的框架，支持 Chrome、Edge、Firefox 和 Safari 浏览器。它简化了扩展的开发过程，提供了统一的 API 和工具，使开发者能够更轻松地创建跨浏览器的扩展。
